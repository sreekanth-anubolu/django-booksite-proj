

1. Ensure install django

2. Create Django Project

    >>> django-admin startporject PROJ_NAME

    PROJ_NAME
        PROJ_NAME
            asgi.py (DO NOT MODIFY THIS FILE)
            wsgi.py (DO NOT MODIFY THIS FILE)

            settings.py
            urls.py

        manage.py (DO NOT MODIFY THIS FILE)


    Test the creation of project
    python manage.py runserver
    # if you see a django page in 127.0.0.1:8000, meaning app/server is running.

3. Create an Application
     >>> cd PROJ_NAME (where manage.py file exists)
     >>> python manage.py startapp APP_NAME

     APP_NAME
        migrations (Folder -> Database related stuff, generated by Django)

        admin.py
        apps.py
        models.py
        tests.py
        views.py

4. Register the Created App in Settings.py
       Add APP_NAME in INSTALLED_APPS List

5. Create urls.py in the application(APP_NAME) that is created
    APP_NAME
        migrations (Folder -> Database related stuff, generated by Django)

        admin.py
        apps.py
        models.py
        tests.py
        views.py

        urls.py (CREATE THIS)

6. Add the URL in the urls.py created under the application (APP_NAME) and mention the respective
    function to be called on URL path

7. Under the Project there is another urls.py
    -> Register the Applications urls here in the urlpatterns list
        ex:
        from django.urls import path, include

        urlpatterns = [
            path('admin/', admin.site.urls),
            # Add the APP URLS, ex: authors is an application name and urls exists in the application folder.
            # authors/ is the base url for authors app...
            path('authors/', include('authors.urls')),
            path('app2/', include('app2.urls')),
            path('app3/', include('app3.urls'))
        ]




REGISTRING TEMPLATES

Under setting.py,
    TEMPLATES
        DIRS
            ADD PATH TO THE TEMPLATES
            ex: 'DIRS': [
                os.path.join(BASE_DIR, 'templates')
            ]
Ensure add all the html files under templates folder in the project directory.
    PROJ_NAME
        templates
            ALL HTML FILES


SETTING DATABASE IN DJANGO

Under Settings.py -> DATABASES -> Set the DB details
    ex:
    DATABASES = {
    'default': {
            'ENGINE': 'django.db.backends.postgresql',
            'NAME': 'booksite',  # Database Name -> Should already be created.
            'USER': 'postgres',
            'PASSWORD': 'Aruba@123',
            'HOST': '127.0.0.1',
            'PORT': '5432',
        }
    }


Creating Models:

    ModelClass should inherit from models.Model

    We can define all the attributes in the ModelClass
    All these attributes are columns in the respective table in the DB.


To Create Tables in the DB

    >>> python manage.py makemigrations
        -> this will create a migrations files under the apps Migration folder.

    >>> python manage.py migrate
        -> this will connect to DB and create the Table or Update the table
